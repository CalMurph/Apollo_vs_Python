=====================================
 Apollo vs Python - Read Me
=====================================

ABOUT THE GAME
--------------
Apollo vs Python is a 2D educational platformer built in the Godot Engine.
You take control of Apollo, exploring levels inspired by Greek mythology while solving
Python programming puzzles hidden in ancient scrolls. Solve the puzzles to
progress each level and ultimately defeat the serpent Python in a final boss fight.

CONTROLS
--------
Arrow Keys or A and D  - Move
Space, W or UP on Arrow Keys - Jump
S or DOWN on arrow keys - Attack with bow
R - Interact with scrolls

LEARNING OUTCOMES
-----------------
The game has five levels, and you will be taught the following Python concepts:
- Variables
- Print statements
- Lists and Dictionaries
- If/elif/else conditionals, and operators
- for and while loops
- Functions

HOW TO PLAY
-----------
1. Extract the zip file to a folder on your computer.
2. Load the folder onto the Godot engine and press the play icon in top right corner
3. Play with keyboard controls as listed above.
